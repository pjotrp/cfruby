Public Class methods

find(basedir, options = {}) {|filename| ...}

Find files that conform to a set of options within a given base directory and yield them to a given block. In addition to all of the options available to FileMatch the following options may be given:

:depth:the maximum depth to recurse into the base directory
:recursive:recurse (default to a depth => 200)
:nonrecursive:don‘t recurse at all (equivilant to :depth => 0)
:followsymlinks:if true symlinks are expanded and returned as their realpath, otherwise they are returned as is (default false)
:returnorder:if unset files are returned in traversal order, if set to ‘delete’ files are returned first followed by directories
:filematch:normally internal use only, allows passing in of a FileMatch object (all other file matching options ignored)

Returns the number of files found
