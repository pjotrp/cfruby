Class Cfruby::Users::UserInfo
In: users.rb  (CVS)
Parent: OpenStruct

A holding object for information about a user. Currently implemented as a simple OpenStruct. Should provide accessor methods for as many of the following as possible (listed in order of importance):

username:username of the user (root)
uid:user id of the user (0)
gid:group id of he user (primary) (0)
fullname:full name of the user (Charlie Root)
shell:default shell assigned to the user
homedir:home directory of the user
