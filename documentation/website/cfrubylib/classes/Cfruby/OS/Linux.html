lookup(key)

returns the value of the given key for this OS. At a minimum an OS should provide the following:

name:returns the name of the OS
version:the version of the OS
osname:e.g. freebsd, linux, windows, etc - these should return true and be case insensitive to allow 
   lookups of the form lookup('Windows')

In addition to the above name and version should be implemented as getter methods for convenience
